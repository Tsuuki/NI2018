## Prérequis

 Il faut avoir installé **Node.js**.
 
 
## Installation

* Clonez le repository. 
* Déplacez-vous dans le répertoire RoBro. 
* Exécutez la commande : `npm install` 
* Exécutez la commande : `npm start` 
* Allez à l'adresse `localhost:4200`
* Enjoy !
